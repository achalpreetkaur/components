Sample Text Box 1

This is an animated text input field which changes its labels position when clicked upon. This element is based on CSS and HTML5 only. No JS/JQuery is being used.

HTML

<span class="input input--jiro">
<input class="input__field input__field--jiro" type="text" id="input-10" />
<label class="input__label input__label--jiro" for="input-10">
<span class="input__label-content input__label-content--jiro">Enter Text</span>
</span>

CSS
