Sample Button 1

Login
This is an animated button that changes its colors when hovered upon. This is based entirely on HTML5 and CSS. No JS/JQuery is being used.

HTML

<a rel="nofollow" rel="noreferrer" href="#" class="btn btn-sm animated-button victoria-two">Login</a>

CSS
